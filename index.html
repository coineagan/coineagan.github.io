_apps

Kaida

A writing assisting WYSIWYG editor created with Rails and React. The app uses text linters and Draft.js to wrap words in decorators with helpful tips.

Minburu

Minburu is a note app that lets you bi-directionally link notes, and returns those notes to you for review using a spacing effect algorithm.

Disengage

Disengage started as a Ruby file that took data from an Excel spreadsheet and automated data entry using Selenium.

It evolved into an Electron application that replaced Excel with a built-in table, and Selenium and Ruby with WebDriverIO and JavaScript. Executables for Windows, MacOS, and Linux were re-built on updates using GitHub actions.

Readerwax

A matchmaking app to connect authors and reviewers. The app used Elasticsearch to connect users with similar interests and let them upload and download e-reader files using Amazon S3.

sharkduck

A spaced repetition flashcard app. The flashcards support a bunch of different content types and study modes. It lets you personalize the algorithm and view progress with statistics.
