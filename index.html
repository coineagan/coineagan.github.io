Sean

Some apps and games I've made

Inkcap

This was made during a hackathon at Fetch. It's a Rails app and a Python service that uses the Playwright MCP to auto-heal broken tests. When you ask it to heal a test, it will rewrite the test and rerun the pipeline until it passes.

LAHIPPO

A mini-project at One Degree. It's an assessment using One Degree resource data to find homeless services in Los Angeles. This was made with Rails and Stimulus.

Implausible

Another One Degree project. It's Cypress code connected to a Rails app that gathers and displays website performance data for specific user roles.

Kaida

A writing assisting WYSIWYG editor made with Rails and React.

Minburu

A note app that bi-directionally links notes and returns those notes for review using a spacing effect algorithm. This was made with Rails and ActionText.

Deckbuilder

A prototype of a deck building card game inspired by Shadowrun:Crossfire and Dragonfire. This was made with HTML, CSS, and my own interactive (not reactive) JavaScript framework.

Hamingja

A little habit app with just enough data to be useful. This was made with Rails and Hotwire Turbo.

Disengage

An Electron application to enter data into a Salesforce system using WebDriverIO. Executables for Windows, MacOS, and Linux were re-built on updates using GitHub actions. The excel-ish design was made to vibe with other tech the clients used.

Readerwax

A matchmaking app to connect authors and reviewers.

sharkduck

A spaced repetition flashcard app. The flashcards support a bunch of different content types and study modes. It lets you personalize the algorithm and view progress with statistics.

Syncing Fast

A data integration tool using the Shopify and QuickBooks API to sync orders.

Heretical

A CSS framework without classes, ids, or sneaky JavaScript. Heretical works by abusing reimagining how six HTML elements should be used.
