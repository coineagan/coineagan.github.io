Example projects

Kaida

Ruby on Rails, React, HTML, CSS

A writing assisting WYSIWYG editor created with Rails and React. The app uses text linters and Draft.js to wrap words in decorators with helpful tooltips. You can also double-click words for a list of synonyms. Other features include timers, auto-saving, and themes.

Minburu

Ruby on Rails, JavaScript, HTML, CSS

Minburu is an editor designed to make idea generation more efficient. It lets you bi-directionally link notes, and returns those notes to you for review using a spacing effect algorithm.

Unlike other spaced repetition software that wants efficient learning, Minburu wants efficient re-learning. Rather than show information right before you forget it, it shows you information right after youâ€™ve forgotten it.

Disengage

Selenium, WebDriverIO, JavaScript, Ruby, Electron, HTML, CSS

Disengage started as a Ruby file that took data from an Excel spreadsheet and automated data entry using Selenium.

It evolved into a full-fledged Electron application that replaced Excel with a built-in table, and Selenium and Ruby with WebDriverIO and JavaScript. Executables for Windows, MacOS, and Linux were made with electron-builder and GitHub actions.

Readerwax

Ruby on Rails, Elasticsearch, Custom Analytics, JavaScript, HTML, CSS

A matchmaking app connecting authors and reviewers. The app used Elasticsearch to filter descriptions to help authors find good matches. Users can upload and download e-reader files using Amazon S3. You can also write pitches and reviews in an in-app Markdown editor.

sharkduck

JavaScript, HTML, CSS

A spaced repetition flashcard app. The flashcards support a bunch of different content types and study modes. It lets you personalize the algorithm and view progress with statistics.
