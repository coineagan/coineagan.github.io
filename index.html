Sean

Some apps and games I've made

Implausible

This was made for One Degree. It is Cypress code connected to a Rails app that gathers and displays website performance data.

LAHIPPO

An assessment using One Degree resource data to find homeless services in Los Angeles. This was made with Rails and Stimulus.

Kaida

A writing assisting WYSIWYG editor made with Rails and React.

Minburu

A note app that bi-directionally links notes and returns those notes for review using a spacing effect algorithm. This was made with Rails and ActionText.

Deckbuilder

A single-player deck building card game inspired by Shadowrun:Crossfire and Dragonfire. This was made with HTML, CSS, and my own JavaScript framework.

Hamingja

A habit app with just enough data to be useful. This was made with Rails and Hotwire Turbo.

Disengage

An Electron application to enter data into a Salesforce system using WebDriverIO. Executables for Windows, MacOS, and Linux were re-built on updates using GitHub actions.

Readerwax

A matchmaking app to connect authors and reviewers. The app used Elasticsearch to connect users with similar interests and let them upload and download e-reader files using Amazon S3.

sharkduck

A spaced repetition flashcard app. The flashcards support a bunch of different content types and study modes. It lets you personalize the algorithm and view progress with statistics.

Syncing Fast

A data integration tool using the Shopify and QuickBooks API to sync orders.

Heretical

A CSS framework without classes, ids, or JavaScript. Heretical works by abusing reimagining how six HTML elements should be used.
